Swiss Matcher
==

Given the state of a Swiss match (can be Pairs or Teams) returns a possibly nil set of matches based on the selection criterion.

Install
==

To install

    git clone https://github.com/njhammond/swiss_matcher
    cd swiss_matcher
    bundle install
